Cloudflare Vectorize Store
==========================

Provides [Cloudflare Vectorize](https://developers.cloudflare.com/vectorize/) vector store integration for Symfony AI Store.

 * [Delete by IDs](https://developers.cloudflare.com/api/resources/vectorize/subresources/indexes/methods/delete_by_ids/)
